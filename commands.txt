# ========================================
# TROUBLESHOOTING
# ========================================
#
# Development server won't start:
# - Check Node.js version: node --version (should be 18+)
# - Delete node_modules and reinstall: npm run clean:install
# - Check .env.local file exists
# - Check for port conflicts: lsof -i :3000 (Mac/Linux) or netstat -ano | findstr :3000 (Windows)
#
# Build fails:
# - Check for TypeScript errors: npm run type-check
# - Check for linting errors: npm run lint
# - Clear .next folder: npm run clean
# - Check environment variables are set correctly
#
# Type errors:
# - Run type check: npm run type-check
# - Check tsconfig.json configuration
# - Ensure all dependencies are installed
#
# Module not found:
# - Reinstall dependencies: npm install
# - Check import paths are correct
# - Verify package is in package.json
#
# Styling issues:
# - Check styles are imported in layout or page
# - Verify CSS classes match component usage
# - Check for conflicting styles
#
# API connection issues:
# - Verify NEXT_PUBLIC_API_BASE_URL in .env.local
# - Check backend server is running
# - Verify CORS settings on backend
# - Check browser console for errors
#
# ========================================
# ENVIRONMENT VARIABLES
# ========================================
#
# Required variables (see .env.local):
# - NEXT_PUBLIC_API_BASE_URL: Backend API base URL
