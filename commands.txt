# NexusCRM - Development Commands

## Setup & Installation
npm install                    # Install all dependencies
npm install <package>          # Install a specific package
npm install -D <package>       # Install as dev dependency

## Development
npm run dev                    # Start development server (http://localhost:3000)
npm run build                  # Build for production
npm run start                  # Start production server
npm run lint                   # Run ESLint

## Database & Backend Services
# Supabase CLI (if using)
npx supabase init              # Initialize Supabase project
npx supabase start             # Start local Supabase
npx supabase stop              # Stop local Supabase
npx supabase db reset          # Reset local database

## Type Checking
npx tsc --noEmit               # Type check without emitting files
npx tsc --watch                # Watch mode type checking

## Code Quality
npm run lint                   # Lint code
npm run lint -- --fix          # Fix linting errors automatically

## Environment Variables
# Create .env.local file with:
# NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
# NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
# NEXT_PUBLIC_GEMINI_API_KEY=your_gemini_api_key
# API_KEY=your_gemini_api_key
# NEXT_PUBLIC_API_BASE_URL=http://107.21.188.21/api

## Git Commands
git status                     # Check git status
git add .                      # Stage all changes
git commit -m "message"        # Commit changes
git push                       # Push to remote
git pull                       # Pull latest changes
git branch                     # List branches
git checkout -b feature/name   # Create and switch to new branch

## Testing & Debugging
npm run dev                    # Start dev server with hot reload
# Open browser DevTools (F12) for debugging
# Use React DevTools extension for component inspection

## Build & Deploy
npm run build                  # Create production build
npm run start                  # Run production build locally
# Check .next/ folder for build output

## Utility Commands
npx next-info                  # Display Next.js environment information
npx create-next-app@latest     # Create new Next.js project (reference)
rm -rf .next                   # Clear Next.js cache (Windows: rmdir /s .next)
rm -rf node_modules            # Remove node_modules (Windows: rmdir /s node_modules)

## Package Management
npm outdated                   # Check for outdated packages
npm update                     # Update packages
npm audit                      # Security audit
npm audit fix                  # Fix security vulnerabilities

## Code Generation (Next.js)
# Create new page: Create app/[route]/page.tsx
# Create new component: Create components/[ComponentName].tsx
# Create new hook: Create hooks/use[HookName].ts
# Create new service: Create services/[serviceName].ts
